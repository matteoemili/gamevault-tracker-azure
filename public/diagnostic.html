🔍 Azure Table Storage Connection Test

This page helps diagnose connection issues with Azure Table Storage.

Configuration

1. Test CORS Configuration

Checks if CORS is properly configured for this domain.

2. Test Authentication

Verifies that your SAS token is valid and has correct permissions.

3. Test Query Operation (Read)

Attempts to read entities from the table.

4. Test Upsert Operation (Write)

Attempts to write a test entity to the table.

5. Performance Test

Measures latency for Azure Table Storage operations.

6. Full Integration Test

Simulates adding and retrieving games like the real app.
